This project implements a real-time facial emotion detection system using deep learning, OpenCV, and Python. It uses a pre-trained Convolutional Neural Network (CNN) model trained on the FER-2013 dataset to classify human emotions such as happy, sad, angry, and neutral from facial expressions captured through a webcam. Face detection is performed using a Haar Cascade classifier, and the detected faces are processed and passed to the trained model for emotion prediction.

The system works in real time by capturing video frames, detecting faces, and displaying the predicted emotion label on the screen. Due to size limitations, the trained .h5 model file is provided via an external download link. This project demonstrates the integration of computer vision and deep learning techniques to build an interactive and practical emotion recognition application.
